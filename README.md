# JL Environmental Consulting Website
Official website for **JL Environmental Consulting**, showcasing environmental consulting scientific expertise, services, projects, and contact information.


## Overview
This website presents:
- Home page
- About page
- Projects portfolio
- Contact page

This website also include: 
- Responsive design for desktop and mobile
- Modern animations using GSAP
- Clean and minimalist interface


## Technologies
- HTML5
- CSS3
- JavaScript
- GSAP (GreenSock Animation Platform)

## Installation
Clone the repository:
```bash
git clone https://github.com/Jennifer-Laurent/JL-Environmental-Consulting.git
```

Open the project folder:
```bash
cd JL-Environmental-Consulting
```

Launch the website by opening:
```
index.html
```
or use the **Live Server** extension in Visual Studio Code.


## Deployment
The website is deployed using **GitHub Pages**.
